## 2026-04-25 - Focus and ARIA fixes\n**Learning:** Raw `<button>` tags and custom accordion toggles in Next.js apps often miss accessibility attributes and keyboard focus styles that are normally provided by Radix UI primitives.\n**Action:** Always check interactive elements that aren't using Radix/ui components for `focus-visible` classes, `aria-label` for icon-only buttons, and `aria-expanded`/`aria-controls` for disclosures.
